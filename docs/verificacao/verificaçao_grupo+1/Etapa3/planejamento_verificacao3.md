## Introdução
A etapa de verificação está presente no projeto pois permite avaliar se os artefatos têm conformidade com o que se espera de cada um deles. Assim, esse é o planejamento para uma avaliação detalhada tanto de conteúdo como da estrutura dos artefados planejados para a etapa 3.

## Objetivo
O objetivo principal deste documento de planejamento de verificação é assegurar que os artefatos produzidos durante a etapa 3 pelo grupo 5 estejam em conformidade com os requisitos especificados e atendam aos critérios de qualidade estabelecidos para o projeto. 

## Metodologia 
A avaliação será feita por meio de um checklist
